OTPublisher Class Reference

Inherits from NSObject
Declared in OTPublisher.h

Overview

A publisher captures an audio-video stream from the device’s microphone and camera. You can then publish the audio-video stream to an OpenTok session by sending the [OTSession publish:] message.

Setting publisher device configuration

  •   cameraPosition

    The preferred camera position. When setting this property, if the change is possible, the publisher will use the camera with the specified position. If the publisher has begun publishing, getting this property returns the current camera position; if the publisher has not yet begun publishing, getting this property returns the preferred camera position.

         property

name

A string that will be associated with this publisher’s stream. This string is displayed at the bottom of publisher videos and at the bottom of subscriber videos associated with the published stream. You can set this name after initializing the publisher and before sending the [OTSession publish:] message. Setting the property after sending the [OTSession publish:] message has no effect on the name displayed for the stream.

@property (nonatomic, retain) NSString *name

Discussion

Note that you can also set the name when you send the [OTPublisher initWithDelegate:name:] message.

This value defaults to an empty string.
